# 接口：chooseTerm  [返回](../README.md)
用例： [选择学期](../用例/选择学期.md)

- 功能：
    老师和学生选择学期并且查看每个学期的课程和实验成绩。
    
- 权限：    
    老师/学生。
    
- API请求地址： 
     http://202.115.82.8:1522/v1/api/chooseTerm/

- 请求方式 ：
    GET

- 请求参数说明:        


  
    
- 返回实例：

        {         
            "status": true,
            term{
                "2015-2016第一学期"
                "2015-2016第二学期"
                "2016-2017第一学期"
                "2016-2017第二学期"
                "2017-2018第一学期"
                "2017-2018第二学期"
            }
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |term|学期|
